JstMStudio

Un estudio polivalente en Asia

JstMStudio es un pequeño estudio fundado por Jst Tan. JstMStudio se enfoca en muchas cosas, desde música, escritura de historias, desarrollo de sitios web y mucho más. JstMStudio es un pequeño estudio independiente remoto que se enfoca en muchas cosas para construir un futuro mejor.



Proyectos

Héroes adolescentes

En 2100, hay dos héroes gemelos chinos adolescentes, llamados Li Hui y Hu Jie con sus padres en Perle, ¡en una aventura para proteger su poderosa ciudad! Teenager Heroes es una serie de libros de ciencia ficción y aventuras, una serie de libros centrada en el futuro de la ciencia y las aventuras. La serie presentaba a los personajes principales, Li Hui y Hu Jie, los gemelos chinos en una increíble aventura en su ciudad, Perle.

InfoHQ

InfoHQ es un sitio web de enciclopedia de contenido gratuito impulsado por colaboradores de todo el mundo para editar y compartir información de todo el mundo. InfoHQ se inspiró en "wiki" de MediaWiki. Actualmente, InfoHQ está disponible para ser alojado en la mayoría (si no en todas) las empresas de alojamiento web gratuitas y de pago en Internet.

Aplicación JstMStudio

La aplicación JstMStudio es una aplicación multiplataforma multipropósito de código abierto creada por JstMStudio. La aplicación JstMStudio actualmente contenía muchas funciones como transmisión de video, tareas pendientes y mucho más de forma gratuita y sin tarifas ocultas. La aplicación JstMStudio es totalmente gratuita para que todos puedan ver nuestros códigos fuente en nuestro Github.



Beneficios de usar JstMStudio

Código abierto

Los proyectos JstMStudio son completamente de código abierto para que todos puedan ver, estudiar y usar nuestros códigos fuente bajo licencias copyleft de código abierto AGPL 3.0 en nuestra organización de Github aquí. Todo lo que se usa para crear el sitio web y la plataforma de JstMStudio es de código abierto para que todos puedan ver el código fuente de forma gratuita. Actualmente no tenemos ni tenemos planes de software de código cerrado también.

Respetuoso con la privacidad

JstMStudio también prioriza la privacidad de nuestros usuarios en todos nuestros proyectos y sitios web para que los usuarios puedan usar nuestras funciones y proyectos de manera segura sin filtrar su privacidad. También recopilamos menos datos de usuarios de nuestros proyectos que cualquier otra empresa y no compartimos ni vendemos ninguno de los datos recopilados por nosotros, protegiéndolos todos en cifrado de alta calidad y no creamos un perfil sobre todos y cada uno de nuestros usuarios.

Ambiente Seguro

Aquí en JstMStudio, ofrecemos el primer entorno más seguro en la World Wide Web (WWW). Moderamos estricta y estrictamente todos y cada uno de los contenidos que se colocan en nuestra plataforma y sitio web por precaución de seguridad para nuestros usuarios. También tenemos unas Directrices comunitarias estrictas que aplicamos a nuestra comunidad para mantenerla segura y limpia. Todos nuestros administradores y moderadores son administradores de élite capacitados para garantizar el mejor entorno en JstMStudio.
