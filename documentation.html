Acerca de

JstMStudio es un pequeño estudio fundado por Jst Tan. JstMStudio se enfoca en muchas cosas, desde música, escritura de historias, desarrollo de sitios web y mucho más. JstMStudio es un pequeño estudio independiente remoto que se enfoca en muchas cosas para construir un futuro mejor.

Contacto

¿Quería ponerse en contacto con JstMStudio? Actualmente, el equipo ofrece múltiples formas para que todos se comuniquen con el equipo. Póngase en contacto con nosotros para obtener información importante, como soporte, comentarios, sugerencias e informes de errores, y no para contenido no importante como el correo no deseado. Somos un equipo pequeño, por lo que es posible que estemos ocupados provocando una respuesta tardía del equipo, así que tenga paciencia. Además, solo respondemos durante nuestro horario comercial oficial.

Contribuyendo a JstMStudio

Hay muchas cosas con las que podrías ayudar y contribuir a los proyectos de JstMStudio y ayudar al twam. Agradecemos todas y cada una de sus ayudas y contribuciones a nuestros proyectos, sin importar si se trata de un pequeño error tipográfico o un error del equipo.



Informar errores

Una forma de contribuir al proyecto sin conocimientos técnicos es informar de los errores al equipo. Si encontró un error en nuestro sitio web o software, cree un problema de Github o envíenos un correo electrónico e infórmelo. Nos ayudaría mucho mejorar nuestro software para que todos pudieran usarlo sin problemas.

Si pudiera, también podría ayudarnos a solucionar los errores informados cuando aún no los hayamos solucionado. Sin duda, ayudaría mucho al equipo a ahorrar tiempo y esfuerzos para encontrar errores. Después de finalizar la programación, puede enviar una solicitud de incorporación de cambios para que uno de nuestros voluntarios la apruebe e implemente también.



Agregar contenido al software JstMStudio

Puedes contribuir al proyecto agregando algún contenido a nuestro software. Puede enviar sitios o proyectos a nuestro motor de búsqueda, agregar videos, música y libros a nuestra biblioteca de contenido para que todos disfruten, etc.



Códigos

Si es desarrollador, también podría ayudarnos a mejorar nuestro software, como mejorar nuestra interfaz de usuario, ayudar a desarrollar y crear nuestras próximas funciones y proyectos nuevos con el equipo, etc. También podría ayudar y contribuir a crear más funciones para nuestros servicios actuales.

Donar

JstMStudio está financiado por personas como usted. No vendemos ningún dato de usuario, no enviamos spam a su pantalla con anuncios ni bloqueamos cosas en el muro de pago. No lo hacemos y creemos que es molesto. Nos financiamos a través de sitios web de venta y donaciones. Si desea apoyar el proyecto, por favor done uno o dos dólares. Puede significar menos para usted, pero significa mucho para nosotros.

Actualmente ofrecemos dos métodos para que todos nos ayuden, apoyen y donen. Puede donarnos a través de Cómprame un café y Ko-Fi. También puede apoyarnos donando una vez o mensualmente.

Extensión

JstMStudio Profile es el primer sitio social que realmente brinda soporte para la extensión en el perfil y las publicaciones de los usuarios. Una extensión es un complemento de software que se instala en un programa, mejorando sus capacidades. Los usuarios pueden elegir entre una variedad de extensiones para usar en su perfil y publicar en la biblioteca de extensiones de perfil de JstMStudio. La extensión JstMStudio Profile está creada por el equipo de personal oficial y la comunidad.

Los usuarios pueden instalar todos los complementos aquí en su código agregándolos antes de que finalicen los elementos de la publicación o en su perfil. Instale el complemento copiando y pegando el código a continuación.



Aplauso

Clap es una de las extensiones oficiales del perfil JstMStudio. Podría usarse en la publicación de usuarios del perfil de JstMStudio. Los usuarios pueden aplaudir a las publicaciones de los usuarios cuando se instala este complemento.



Comentario

Comment es una de las extensiones oficiales del perfil de JstMStudio. Los usuarios pueden agregarlos a su publicación para que el espectador los comente. Los usuarios también pueden usarlo para obtener comentarios sobre sus publicaciones.



Me gusta

Me gusta es una de las extensiones oficiales del perfil de JstMStudio. Los usuarios pueden agregarlos a su publicación para que el espectador y el escritor muestren su reacción a la publicación. Los usuarios también podrían reaccionar para mostrar su apoyo.



Compartir

Share es una de las extensiones oficiales del perfil de JstMStudio. Los usuarios pueden agregarlos a su publicación para que el espectador comparta su publicación en las redes sociales. Los usuarios también podrían reaccionar para mostrar su apoyo compartiendo la publicación.



Arriba-abajo

Updown es una de las extensiones oficiales del perfil JstMStudio. Los usuarios pueden agregarlos a sus publicaciones para que los espectadores y escritores muestren sus reacciones a la publicación. Los usuarios también podrían reaccionar para mostrar su apoyo. Updown también podría usarse para encuestas.

API

JstMStudio proporcionó muchos servicios y características dentro de nuestro software de plataforma, JstMStudio App Platform. JstMStudio API es una interfaz de programación de aplicaciones para la plataforma de aplicaciones JstMStudio que brinda soporte para que todos los desarrolladores implementen servicios como en la plataforma de aplicaciones JstMStudio en su sitio web sin desarrollar las funciones en sí mismos de forma totalmente gratuita y sin tarifas. Está diseñado para que todos lo usen y no para ganar dinero de usuarios como el desarrollador y el propietario del sitio web.

JstMStudio API es una API de código abierto (interfaz de programación de aplicaciones). Todos podían ver, leer y estudiar todos los códigos fuente de la API de JstMStudio en nuestro repositorio oficial de Github. La API de JstMStudio está construida usando HTML, CSS y JavaScript estándar a diferencia de cualquier API tradicional y tiene licencia bajo AGPL 3.0.
